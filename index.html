ackoctl

CLI for aerospike-cluster-manager. Source: aerospike-ce-ecosystem/ackoctl.

APT and YUM repositories on this domain are deprecated. The signed apt/ and yum/ repositories under aerospike-ce-ecosystem.github.io/ackoctl/ stopped receiving updates as of v0.2.0 and will be removed after v0.3.0. Existing installations will continue to resolve cached v0.1.x packages until then. Migrate to the shell installer below, then drop any old sources.list.d/ackoctl.list / yum.repos.d/ackoctl.repo entries.

Linux & macOS — shell installer

curl -fsSL https://raw.githubusercontent.com/aerospike-ce-ecosystem/ackoctl/main/install.sh | sh

Detects OS/arch (darwin/linux × amd64/arm64), verifies the sha256 from checksums.txt, installs to /usr/local/bin/ackoctl (falls back to ~/.local/bin when not writable). Pin with ACKOCTL_VERSION=v0.2.0.

Homebrew (macOS)

brew install aerospike-ce-ecosystem/tap/ackoctl

Self-update

ackoctl upgrade            # in-place update to the latest release
ackoctl upgrade --check    # report current vs latest, do not install

Migrating off apt / dnf

If you previously installed via the APT or YUM repositories, run the installer above, then remove the deprecated sources:

# Debian / Ubuntu
sudo rm -f /etc/apt/sources.list.d/ackoctl.list /etc/apt/keyrings/ackoctl.gpg
sudo apt update

# RHEL / Fedora / Rocky
sudo rm -f /etc/yum.repos.d/ackoctl.repo
